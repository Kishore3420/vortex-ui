# Animation Page Template Structure

This template should be followed for all animation documentation pages.

## Required Sections

### 1. **Hero Section**

- Title
- Brief description (1-2 sentences)
- Visual example (live demo)

### 2. **Overview**

- What the animation does
- When to use it
- Key characteristics

### 3. **How It Works**

- Technical explanation
- Initial state
- Animation process
- Trigger mechanism
- Code flow explanation

### 4. **Props Reference**

- Complete props table
- Type definitions
- Default values
- Detailed descriptions
- Usage examples for each prop

### 5. **Basic Usage**

- Simple code example
- Step-by-step guide
- Import statement
- Minimal working example

### 6. **Interactive Examples**

- Live demos showing different variations
- Different prop combinations
- Visual comparisons

### 7. **Use Cases in Web Design**

- Where to use in real websites
- Design patterns
- Best practices
- Common scenarios

### 8. **Real-World Examples**

- Complete code examples
- Real website scenarios
- Production-ready snippets
- Multiple variations

### 9. **Customization Guide**

- Advanced usage
- Combining with other animations
- Custom easing
- Performance tweaks

### 10. **Performance & Accessibility**

- Performance considerations
- GPU acceleration notes
- Accessibility (prefers-reduced-motion)
- Browser support

### 11. **Tips & Best Practices**

- Common mistakes to avoid
- Optimization tips
- Design recommendations

## Content Guidelines

- **Learning-Focused**: Explain the "why" not just the "how"
- **Visual**: Include live examples for every concept
- **Practical**: Show real-world usage, not just theory
- **Complete**: Cover all props and variations
- **Accessible**: Consider all users and devices
